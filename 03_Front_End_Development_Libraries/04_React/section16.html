16 - Use Default Props

The code editor shows a ShoppingCart component. Define default props on this component which specify a prop items with a value of 0.

Starter Code:

const ShoppingCart = (props) => { return ( <div> <h1>Shopping Cart Component</h1> </div> ) }; // Change code below this line

Solution:

const ShoppingCart = (props) => { return ( <div> <h1>Shopping Cart Component</h1> </div> ) }; // Change code below this line ShoppingCart.defaultProps={ items: 0, }
