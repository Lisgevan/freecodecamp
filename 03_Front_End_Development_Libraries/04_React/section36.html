36 - Introducing Inline Styles

Add a style attribute to the div in the code editor to give the text a color of red and font size of 72px.

Starter Code:

class Colorful extends React.Component { render() { return ( <div>Big Red</div> ); } };

Solution:

class Colorful extends React.Component { render() { return ( <div style={{color:"red", fontSize:72/*or "72px"*/}}>Big Red</div> ); } };
