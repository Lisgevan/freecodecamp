21 - Create a Bootstrap Row

Create a div element below the h3 tag, with a class of row.

Starter Code:

<div class="container-fluid"> <h3 class="text-primary text-center">jQuery Playground</h3> </div>

Solution:

<div class="container-fluid"> <h3 class="text-primary text-center">jQuery Playground</h3> <div class="row"> </div> </div>
