34. Use the CSS Transform scale Property to Change the Size of an Element

Increase the size of the element with the id of ball2 to 1.5 times its original size.

Starter Code:

<style> .ball { width: 40px; height: 40px; margin: 50 auto; position: fixed; background: linear-gradient(35deg, #ccffff, #ffcccc); border-radius: 50%; } #ball1 { left: 20%; } #ball2 { left: 65%; } </style> <div class="ball" id="ball1"></div> <div class="ball" id="ball2"></div>

Solution:

<style> .ball { width: 40px; height: 40px; margin: 50 auto; position: fixed; background: linear-gradient(35deg, #ccffff, #ffcccc); border-radius: 50%; } #ball1 { left: 20%; } #ball2 { left: 65%; transform: scale(1.5); } </style> <div class="ball" id="ball1"></div> <div class="ball" id="ball2"></div>
