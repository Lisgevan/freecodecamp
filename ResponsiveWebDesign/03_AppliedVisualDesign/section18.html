18. Adjust the Hover State of an Anchor TagPassed

The code editor has a CSS rule to style all a tags black. Add a rule so that when the user hovers over the a tag, the color is blue.

Starter Code:

<style> a { color: #000; } </style> <a href="https://freecatphotoapp.com/" target="_blank">CatPhotoApp</a>

Solution:

<style> a { color: #000; } a:hover { color: blue; } </style> <a href="https://freecatphotoapp.com/" target="_blank">CatPhotoApp</a>
