26. Learn about Complementary Colors

Change the background-color property of the blue and yellow classes to their respective colors. Notice how the colors look different next to each other than they do compared against the white background.

Starter Code:

<style> body { background-color: #FFFFFF; } .blue { background-color: #000000; } .yellow { background-color: #000000; } div { display: inline-block; height: 100px; width: 100px; } </style> <div class="blue"></div> <div class="yellow"></div>

Solution:

<style> body { background-color: #FFFFFF; } .blue { background-color: blue; } .yellow { background-color: yellow; } div { display: inline-block; height: 100px; width: 100px; } </style> <div class="blue"></div> <div class="yellow"></div>
