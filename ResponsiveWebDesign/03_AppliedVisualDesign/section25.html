25. Center an Element Horizontally Using the margin Property

Center the div on the page by adding a margin property with a value of auto.

Starter Code:

<style> div { background-color: blue; height: 100px; width: 100px; } </style> <div></div>

Solution:

<style> div { background-color: blue; height: 100px; width: 100px; margin: auto; } </style> <div></div>
