22. Lock an Element to the Browser Window with Fixed Positioning

The navigation bar in the code is labeled with an id of navbar. Change its position to fixed, and offset it 0 pixels from the top and 0 pixels from the left. After you have added the code, scroll the preview window to see how the navigation stays in place.

Starter Code:

<style> body { min-height: 150vh; } #navbar { width: 100%; background-color: #767676; } nav ul { margin: 0px; padding: 5px 0px 5px 30px; } nav li { display: inline; margin-right: 20px; } a { text-decoration: none; } </style> <header> <h1>Welcome!</h1> <nav id="navbar"> <ul> <li><a href="">Home</a></li> <li><a href="">Contact</a></li> </ul> </nav> </header> <p>I shift up when the #navbar is fixed to the browser window.</p>

Solution:

<style> body { min-height: 150vh; } #navbar { position: fixed; top: 0px; left: 0px; width: 100%; background-color: #767676; } nav ul { margin: 0px; padding: 5px 0px 5px 30px; } nav li { display: inline; margin-right: 20px; } a { text-decoration: none; } </style> <header> <h1>Welcome!</h1> <nav id="navbar"> <ul> <li><a href="">Home</a></li> <li><a href="">Contact</a></li> </ul> </nav> </header> <p>I shift up when the #navbar is fixed to the browser window.</p>
