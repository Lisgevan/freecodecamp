21. Lock an Element to its Parent with Absolute PositioningPassed

Lock the #searchbar element to the top-right of its section parent by declaring its position as absolute. Give it top and right offsets of 50 pixels each.

Starter Code:

<style> #searchbar {} section { position: relative; } </style> <h1>Welcome!</h1> <section> <form id="searchbar"> <label for="search">Search:</label> <input type="search" id="search" name="search"> <input type="submit" name="submit" value="Go!"> </form> </section>

Solution:

<style> #searchbar { position: absolute; top: 50px; right: 50px; } section { position: relative; } </style> <h1>Welcome!</h1> <section> <form id="searchbar"> <label for="search">Search:</label> <input type="search" id="search" name="search"> <input type="submit" name="submit" value="Go!"> </form> </section>
