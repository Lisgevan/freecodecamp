CSS Flexbox

13. Use the flex-grow Property to Expand Items

Add the CSS property flex-grow to both #box-1 and #box-2. Give #box-1 a value of 1 and #box-2 a value of 2.

Starter Code:

<style> #box-container { display: flex; height: 500px; } #box-1 { background-color: dodgerblue; height: 200px; } #box-2 { background-color: orangered; height: 200px; } </style> <div id="box-container"> <div id="box-1"></div> <div id="box-2"></div> </div>

Solution:

<style> #box-container { display: flex; height: 500px; } #box-1 { background-color: dodgerblue; height: 200px; flex-grow: 1; } #box-2 { background-color: orangered; height: 200px; flex-grow: 2; } </style> <div id="box-container"> <div id="box-1"></div> <div id="box-2"></div> </div>
