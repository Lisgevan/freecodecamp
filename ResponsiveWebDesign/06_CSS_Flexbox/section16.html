CSS Flexbox

16. Use the order Property to Rearrange Items

Add the CSS property order to both #box-1 and #box-2. Give #box-1 a value of 2 and give #box-2 a value of 1.

Starter Code:

<style> #box-container { display: flex; height: 500px; } #box-1 { background-color: dodgerblue; height: 200px; width: 200px; } #box-2 { background-color: orangered; height: 200px; width: 200px; } </style> <div id="box-container"> <div id="box-1"></div> <div id="box-2"></div> </div>

Solution:

<style> #box-container { display: flex; height: 500px; } #box-1 { background-color: dodgerblue; order: 2; height: 200px; width: 200px; } #box-2 { background-color: orangered; order: 1; height: 200px; width: 200px; } </style> <div id="box-container"> <div id="box-1"></div> <div id="box-2"></div> </div>
