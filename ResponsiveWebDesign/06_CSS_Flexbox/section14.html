CSS Flexbox

14. Use the flex-basis Property to Set the Initial Size of an Item

Set the initial size of the boxes using flex-basis. Add the CSS property flex-basis to both #box-1 and #box-2. Give #box-1 a value of 10em and #box-2 a value of 20em.

Starter Code:

<style> #box-container { display: flex; height: 500px; } #box-1 { background-color: dodgerblue; height: 200px; } #box-2 { background-color: orangered; height: 200px; } </style> <div id="box-container"> <div id="box-1"></div> <div id="box-2"></div> </div>

Solution:

<style> #box-container { display: flex; height: 500px; } #box-1 { background-color: dodgerblue; height: 200px; flex-basis: 10em; } #box-2 { background-color: orangered; height: 200px; flex-basis: 20em; } </style> <div id="box-container"> <div id="box-1"></div> <div id="box-2"></div> </div> </div>
